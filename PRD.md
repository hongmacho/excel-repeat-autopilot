# Excel 반복 자동화 마법사 (Excel Repeat Autopilot) — 제품 요구사항서

## 1. 제품 개요

**제품명**: Excel 반복 자동화 마법사 (Excel Repeat Autopilot)  
**타겟 사용자**: 비기술자 소규모 사업주, 회계사, HR 담당자  
**핵심 가치**: 매달 반복되는 Excel/CSV 작업(청구서 생성, 재고 정산, 급여 계산)을 **코드 없이** 웹 UI에서 정의하고 버튼 하나로 자동화

**주요 통증점 (Pain Evidence)**:
- 소규모 사업이 매달 반복 작업으로 **33시간(월 업무의 15%)** 낭비
- No-code 도구들(Zapier/Make)은 복잡하고 개발자식 사고 요구
- Excel 데이터 형식 변동을 일일이 대응하는 번거로움

**핵심 차별점**:
- **Excel/CSV 특화**: 드래그앤드롭으로 열 선택 → 필터/복사/계산 → 저장 및 반복 사용
- **로컬-우선**: 클라우드 저장 없음, 데이터는 브라우저 + SQLite 로컬 저장
- **비기술자 UX**: Zapier 대비 80% 더 간단, 명확한 미리보기로 안전성 확보

## 2. MUST-HAVE 기능 (8개)

1. **파일 업로드**: CSV/Excel 파일 로컬 브라우저 업로드
2. **레시피 정의 UI**: 필터, 열 선택, 복사, 계산, 정렬
3. **미리보기**: 레시피 적용 전 결과물 미리 표시
4. **레시피 저장 및 재사용**: 정의한 레시피를 저장하고 재사용
5. **결과 내보내기**: CSV/Excel 다운로드
6. **레시피 템플릿 라이브러리**: 초기 5개 템플릿
7. **빈 상태 처리 및 에러 안내**: 명확한 메시지
8. **대시보드 / 통계**: KPI 카드, 실행 로그, 월별 추이

## 3. 화면 구조 (7개+)

- S1: 홈 화면
- S2: 레시피 관리 페이지  
- S3: 레시피 생성/편집 페이지
- S4: 레시피 실행 페이지
- S5: 템플릿 라이브러리
- S6: 대시보드 / 통계
- S7: 설정 페이지
- S8: 에러 페이지

## 4. 한국어 UI 텍스트

모든 UI 텍스트는 한국어로 제공됩니다:
- 버튼: "새 레시피 만들기", "변환 시작", "저장"
- 에러: "필수 입력 항목입니다", "열을 찾을 수 없습니다"
- 빈 상태: "아직 만든 레시피가 없습니다"

## 5. 비기능 요구사항

- **성능**: 1MB 파일 < 2초, 10K행 레시피 < 5초
- **보안**: 로컬-only, 외부 API 금지
- **호환성**: Chrome/Edge 최신 2개 버전, Firefox, Safari
- **에러 처리**: 모든 상황에서 명확한 메시지
